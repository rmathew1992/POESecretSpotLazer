Required installations for this project

# OS: Ubuntu

# 1. Download OpenCV and Install OpenCV
- Download archive and extract it
http://opencv.org/downloads.html
- Follow instructions in the following page (It takes long time when doing make)
http://docs.opencv.org/doc/tutorials/introduction/linux_install/linux_install.html

# 2. Install SimpleCV
- Download archive and extract it
http://simplecv.org/download
- CD to extracted directory and just type following command
> sudo python setup.py install
